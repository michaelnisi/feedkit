SQLFormatter

class SQLFormatter

SQLFormatter is a base class for formatters that produce SQL statements from FeedKit structures, creating and transforming SQLite rows into FeedKit core objects.

These formatters should not return explicit transactions, leave that to the call site, where the user knows the context.

Remember to respect SQLite limits when using this class. Some of its functions might exceed the maximum depth of an SQLite expression tree. Here’s the deal: basically every time an array of identifiers is longer than 1000, we have to slice it down.
