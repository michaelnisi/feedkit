FeedCaching
  • fulfill(_:ttl:)

    Queries the local cache for entries and returns a tuple of cached entries and unfulfilled entry locators, if any.

    Remember that entries cannot be stale—call site makes the calls.

    Despite ttl, older entries are returned, however their locators are included as unfulfilled. For groups the timestamp of the latest entry is relevant.

    Throws

    Might throw database errors.

    Parameters

    The selection of entries to fetch.
    ttl

    Return Value

    A tuple of cached entries and URLs not satisfied by the cache.

SearchCaching
  • update(feeds:for:)

    Update feeds associating them with the specified search term, which is also added to the suggestions table of the database.

    If feeds is not empty, we got at least one suggestion for the specified term, therefore we need to update the subcached dictionary accordingly, to make sure this term is not skipped the next time a user is requesting suggestions for this term or its predecessors.

    Throws

    Might throw database errors: various SkullError types.

    Declaration

    Swift

    public func update(feeds: [Feed], for term: String) throws

    Parameters

    feeds

    The feeds to cache.
    term
