Searching

public protocol Searching

The search API of the FeedKit framework.

  • search(_:perFindGroupBlock:searchCompletionBlock:)

    Search for feeds by term using locally cached data requested from a remote service. This method falls back on cached data if the remote call fails; the failure is reported by passing an error in the completion block.

    Declaration

    Swift

    @discardableResult
func search(_ term: String, perFindGroupBlock: ((Error?, [Find]) -> Void)?, searchCompletionBlock: ((Error?) -> Void)?) -> Operation

    Parameters

    term

    The term to search for or a feed URL.
    perFindGroupBlock

    The block to receive finds.
    searchCompletionBlock

    The block to execute after the search is complete.

    Return Value

    The, already executing, operation.

  • suggest(_:perFindGroupBlock:suggestCompletionBlock:)

    Get lexicographical suggestions for a search term combining locally cached and remote data.

    Declaration

    Parameters

    term

    The term to search for.
    perFindGroupBlock

    suggestCompletionBlock
